一、Hexo 是什么（简要）

Hexo 是一个基于 Node.js 的静态博客框架，通过 Markdown 生成静态网页，常用于搭建个人博客或学术主页（如 GitHub Pages）。

官方定义：

A fast, simple & powerful blog framework powered by Node.js.
— Hexo Documentation

📚 官方文档：https://hexo.io/docs/
